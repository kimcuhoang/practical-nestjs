# Practical NestJS

- An example of using [Nest](https://github.com/nestjs/nest)
- High-lighted points
  - Use TypeOrm as ORM within Postgres
  - Use Redis
  - Use [testcontainer](https://testcontainers.com/) to implement e2e tests

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
